# Description

A responsive landing page abount bakery with animations.

---

## Technologies

- HTML
- CSS + SCSS
- JS
- Gulp
- Responsive design
- Scroll Reveal

---

## Launch project

```
npm install - install dependencies
gulp dev - launch the project in development mode
```
